# OpenQilin — Sponsorship and Startup-Credit Program Shortlist

Priority order for applications. Apply to Tier 1 first; Tier 2 in parallel once Tier 1 is submitted.

---

## Tier 1 — Apply immediately (no revenue or investment required)

### Google for Startups Cloud Program — Pre-funded tier
- **What:** $2,000 in Google Cloud credits for pre-funded startups
- **Why OpenQilin qualifies:** Open-source AI project; no funding required for pre-funded tier
- **Apply:** https://cloud.google.com/startup
- **What to submit:** Project description, GitHub link, use-case (LLM inference + cloud hosting)
- **Status:** Not yet submitted

### AWS Activate — Founders tier
- **What:** $1,000 in AWS credits; no investor affiliation required
- **Why OpenQilin qualifies:** Early-stage open-source project; self-funded
- **Apply:** https://aws.amazon.com/startups/lp/aws-activate-credits
- **What to submit:** Company/project info, use-case, estimated AWS spend
- **Status:** Not yet submitted

### GitHub Sponsors
- **What:** Community funding directly through GitHub; starts at any amount
- **Why OpenQilin qualifies:** Open-source, Apache 2.0, active contributor community
- **Apply:** https://github.com/sponsors/deyiwang27 (see `github-sponsors-setup.md`)
- **Status:** Active

---

## Tier 2 — Apply after Tier 1 (may require more materials or traction)

### Google for Startups — AI startup program
- **What:** Up to $350,000 for AI-first startups; higher bar than pre-funded tier
- **Why OpenQilin may qualify:** Governed AI runtime for solopreneurs is a clear AI-native product
- **Apply:** https://cloud.google.com/startup/ai
- **What to submit:** Full application including product thesis, demo, team, roadmap
- **Status:** Prepare after demo (M17-WP3) and website (M17-WP5) are live

### Microsoft for Startups — Azure credits
- **What:** Up to $5,000 in Azure credits
- **Apply:** https://www.microsoft.com/en-us/startups
- **Status:** Low priority (OpenQilin stack is GCP/AWS-aligned); apply if Azure credits are useful

---

## Tier 3 — Longer-term (requires traction or network access)

### Anthropic developer programs
- **What:** No public startup-credits program as of March 2026; monitor for announcements
- **Why relevant:** OpenQilin is designed for Claude-compatible LLM backends
- **Action:** Watch https://anthropic.com for developer program announcements

### Open-source foundation grants (NLnet, Open Collective, etc.)
- **What:** Small grants for open-source infrastructure projects
- **Why relevant:** OpenQilin's governance and policy engine has public-good utility
- **Action:** Evaluate after first public release; apply when traction is visible
