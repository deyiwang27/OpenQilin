# GitHub Sponsors — Setup Guide

This guide walks through activating GitHub Sponsors for the deyiwang27 account so the "Sponsor" button appears on OpenQilin's repository.

**Time required:** 15–30 minutes. GitHub review takes 1–3 business days after submission.

---

## Step 1 — Start the GitHub Sponsors application

1. Go to: https://github.com/sponsors/accounts
2. Click **"Get sponsored"**
3. Select the **deyiwang27** account (personal account, not an organisation)
4. Click **"Join the waitlist"** or **"Apply"** — the exact button depends on your region's availability

---

## Step 2 — Complete your sponsor profile

GitHub will ask for:

- **Display name:** `Deyi Wang`
- **Short bio:** `Building OpenQilin — a governed AI operating system for solopreneurs`
- **Introduction (longer bio):** Use the project summary from `docs/sponsorship/project-summary.md`
- **Featured work:** Link to the OpenQilin repository
- **Goals:** Set a goal (e.g., "Cover LLM API costs for development and demo hosting — $50/month")

---

## Step 3 — Set up sponsorship tiers

Create at least two tiers:

| Tier | Price | Name | Description |
|---|---|---|---|
| 1 | $5/month | Supporter | Support OpenQilin's development. Every contribution helps. |
| 2 | $25/month | Contributor | Sustain active development and keep the demo environment running. |

You can add more tiers later. Start simple.

---

## Step 4 — Connect a payout method

GitHub requires a Stripe account (or equivalent) to receive payments.

1. Follow the "Set up payout" prompt in the GitHub Sponsors dashboard
2. Connect a Stripe account (create one at stripe.com if you don't have one)
3. Verify your identity as prompted

---

## Step 5 — Submit for review

Once the profile and payout are configured, click **"Submit for review"**. GitHub typically reviews within 1–3 business days.

---

## Step 6 — Verify FUNDING.yml is live

Once approved, verify the "Sponsor" button appears on the OpenQilin repository:

1. Go to: https://github.com/deyiwang27/OpenQilin
2. Confirm the **"Sponsor"** button appears in the top-right of the repo header
3. Click it and confirm your profile loads

`FUNDING.yml` is already in the repo pointing to `deyiwang27`. No further code changes needed.

---

## After activation

- Add your GitHub Sponsors link to `README.md` once the profile is live
- Mention it in the first public announcement of OpenQilin
- List GitHub Sponsors status as "active" in the program shortlist (`docs/sponsorship/program-shortlist.md`)
